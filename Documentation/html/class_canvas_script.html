◆ BeginShipPlacement()

void CanvasScript.BeginShipPlacement ( )
inlineprivate
Precondition
Teams will be given a number of ships based on the shipSelector option value and be instaintiated within incremental sizes.
BeginShipPlacement will listen for the Confirm Button onClick event.
Confirm Button is active.
Postcondition
BeginShipPlacement method will disable the Ship Selection Menu and enable the Ship Placement Menu.
Player Grids will be set active for players to began placing their fleets. Grid images will be disabled until ship placement is complete.
Player Fleets will be instantiated for players to place ships into the grids.
Parameters
None.
Returns
None.

◆ Update()

void CanvasScript.Update ( )
inline
Postcondition
Update is called once per frame
Update method will check if all the Team's ships are placed on the board.
Afterwards it will enable the Start Button and the game will proceed.
Update method will check for any ships lost or if a player is lost, and notify the players.
Once A player loses, the Game Over Screen will pop up.
